# Full Book Slide Message Map

## 목적

이 문서는 『부의 게임, 대충 뛰어들어 끝까지 살아남기』를 60분 강연으로 옮기기 위한 슬라이드 메시지 맵이다. 실제 Google Slides 제작 전 단계이며, 각 슬라이드가 어떤 챕터에 연결되는지 먼저 고정한다.

## 60분 버전 흐름

|---:|---|---|---|

## 60분 시간 배분

| 구간 | 시간 | 내용 |
|---|---:|---|
| 오프닝 | 5분 | 로그아웃 방지 게임 프레임 |
| 심리와 목표 | 15분 | 심리적 지연, 머니 스크립트, 북극성 |
| 현재 좌표 | 10분 | 인벤토리, 현금흐름, 기록 |
| 시스템과 투자 | 20분 | 자동화, 복리, 생존, 지수 투자, 자산 배분 |
| 실행 전환 | 10분 | 생애 주기, 벌기의 판, 오늘 10분 행동 |

## 슬라이드 제작 원칙

- 특정 상품 추천, 수익 보장, 공포 마케팅 문장을 넣지 않는다.
- 각 파트는 책의 용어를 그대로 사용하되, 강연에서는 문장을 더 짧게 끊는다.
- 표와 숫자는 10장처럼 원고에서 이미 출처가 붙은 것만 사용한다.
- 청중 실습은 민감한 개인 재무 정보를 현장에 제출하지 않는 방식으로 설계한다.
