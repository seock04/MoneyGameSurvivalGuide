# Coaching Program: 3x 1:1 Semi-Structured

## 대상

- 투자 경험 0~초급
- 정보 과잉/완벽주의로 실행이 멈춘 사람

## 약속(Outcome)

- 목표(북극성)와 현재 좌표(인벤토리)가 한 장으로 정리된다.
- 자동 사냥(자동화)으로 "실행이 굴러가는 구조"가 만들어진다.
- 불안이 올라와도 로그아웃하지 않도록 생존 규칙과 12주 플랜이 생긴다.

## 포함/비포함

- 포함: 목표/현금흐름/재무상태/자동화/리스크 관리/기본 자산 배분 방향
- 비포함: 특정 종목/상품 추천, 수익 보장, 매매 타이밍 예언

## 구조 요약(3회차)

- Session 1: 북극성 + 인벤토리(현재 좌표)
- Session 2: 자동 사냥 시스템(자동화/적립)
- Session 3: 생존/유지(리스크/심리/루틴) + 12주 플랜

## 세션 상세

### Session 1: Direction + Inventory

- Goal: "왜"와 "지금 어디"를 고정한다.
- Flow:
  - 북극성 1문장 만들기
  - SMART 목표 1개 도킹
  - 인벤토리(재무상태/현금흐름) 대충 작성
- Exercises:
  - `worksheet_goal_north_star_smart.md`
  - `worksheet_inventory_cashflow_balance.md`
- Homework:
  - 인벤토리 숫자를 1회 업데이트(대충 -> 조금 더 정확)

### Session 2: Auto-farming System

- Goal: 의지력 대신 구조를 만든다.
- Flow:
  - 자동이체/적립의 최소 설계(2개)
  - "따뜻하고 대충 합리적인" 자산 배분 방향(초안)
  - 지출 누수 1개 제거 계획
- Exercises:
  - `worksheet_system_survival_rules.md`의 자동화 파트 선작성
- Homework:
  - 자동이체 1개 실제 실행(가능하면)

### Session 3: Survival + Maintenance

- Goal: 로그아웃 방지 규칙과 유지 루틴을 만든다.
- Flow:
  - 비상금 기준 확정
  - 하지 않을 것(레버리지 등) 1개 확정
  - 하락장 대응 수칙 3줄 작성
  - 12주 실행 플랜 고정
- Exercises:
  - `worksheet_system_survival_rules.md` 완성
- Homework:
  - 4주 후 점검 일정 캘린더 등록

## 운영 메모

- 세션 사이 행동을 최소화한다: "오늘 할 수 있는 10분짜리 1개"가 기본 단위다.
- 불안이 강한 내담자일수록 "수익률"이 아니라 "수면/평온"을 성과 지표로 둔다.
