# Coaching Program: 3x Group Coaching

## 대상

- 실행이 필요한 초보자/초중급자 그룹
- 서로의 질문을 통해 "나만의 문제"를 "보편적 패턴"으로 바꾸고 싶은 사람

## 약속(Outcome)

- 그룹 기준의 공통 프레임을 얻고, 각자의 최소 실행 플랜을 만든다.
- 서로의 사례를 통해 불안을 줄이고, 실행 장벽을 낮춘다.

## 포함/비포함

- 포함: 목표/인벤토리/자동화/리스크/유지 루틴 설계
- 비포함: 개별 자산의 상세 판단(종목 추천 등), 개인 민감 정보 수집

## 구조 요약(3회차)

- Session 1: 북극성 + 목표(그룹 공유 가능한 수준)
- Session 2: 인벤토리 워크숍 + 자동 사냥 설계
- Session 3: 생존 규칙 + 12주 실행 플랜(그룹 피드백)

## 세션 상세

### Session 1: North Star Workshop

- Flow:
  - 책 프레임 소개(부의 게임/대충 합리)
  - 북극성 1문장 만들기(공유 가능 버전)
  - SMART 목표 1개 작성
- Worksheet:
  - `worksheet_goal_north_star_smart.md`

### Session 2: Inventory + Auto-farming Workshop

- Flow:
  - 인벤토리(대충) 작성
  - 자동이체 2개 설계(모두에게 적용 가능한 기본형)
  - 새는 돈 1개 제거(선택)
- Worksheet:
  - `worksheet_inventory_cashflow_balance.md`
  - `worksheet_system_survival_rules.md`(자동화 파트)

### Session 3: Survival + 12-week Plan

- Flow:
  - 비상금/레버리지 금지/하락장 수칙 정리
  - 12주 플랜 작성 후 짝 피드백
- Worksheet:
  - `worksheet_system_survival_rules.md` 완성

## 운영 메모

- 공유는 원칙적으로 "금액"이 아니라 "비율/원칙/행동" 중심으로 한다.
- 그룹은 '정답'이 아니라 '다양한 합리적 선택지'를 보여주는 장치다.
