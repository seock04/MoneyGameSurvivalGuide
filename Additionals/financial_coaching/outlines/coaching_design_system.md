# Coaching Design System (3-session)

이 문서는 코칭 형태가 달라도 동일하게 재사용되는 "3회차 설계 골격"이다.

## 3회차 골격

- Session 1: 방향과 현재 좌표 고정
  - 북극성/SMART 목표(방향)
  - 인벤토리(재무상태/현금흐름)로 현재 좌표 확인
- Session 2: 자동 사냥 시스템 설계
  - 자동이체/적립(시스템)
  - 자산배분은 "따뜻하고 대충 합리적인" 수준부터 시작
- Session 3: 생존과 유지(로그아웃 방지)
  - 비상금/레버리지 금지/리밸런싱 루틴
  - 심리(불안) 대응 수칙 + 12주 실행 플랜

## 산출물(각 포맷 공통)

- 프로그램 설계 문서 1개
- 세션별 슬라이드 아웃라인 3개(또는 그룹 운영용 3개)
- 워크시트 3종(목표/인벤토리/시스템-유지)
