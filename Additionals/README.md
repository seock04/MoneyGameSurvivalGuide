# Additionals

책 외 콘텐츠는 이 디렉터리에서 관리한다.

반드시 먼저 확인:
- `RULES.md`

## Structure
- `financial_lectures/` : 재무 강의 자료
- `financial_coaching/` : 재무 코칭 자료
- `blog_posts/` : 블로그 포스팅 자료
- `content_marketing/` : 콘텐츠 마케팅 자료

각 폴더는 다음 트리를 공통으로 사용한다.

```text
Additionals/<domain>/
├── outlines/
├── drafts/
├── final/
├── assets/
└── references/
```
