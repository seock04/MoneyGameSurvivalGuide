# 6. 당신의 인벤토리를 열어보세요: '대충'이라도 적어보는 현재 좌표 (현금흐름과 재무상태)

[체크인 질문]
• 지금 당장 지갑과 통장에 있는 돈을 다 합치면 얼마인지, 1분 안에 대략이라도 말할 수 있는가?
• 매달 고정적으로 나가는 돈(숨만 쉬어도 나가는 돈)이 정확히 얼마인지 알고 있는가?
• 당신의 총자산 중에서 '진짜 내 돈'과 '은행에서 잠시 빌린 돈'의 비율은 어느 정도인가?

---

## 🎒 모험을 떠나기 전, 가방부터 열어보자
앞 장에서 우리는 나만의 '북극성(목표)'을 찾았다. 목적지를 정했으니 이제 출발할 차례다. 그런데 잠깐, 출발하기 전에 꼭 해야 할 일이 하나 더 있다. 바로 지금 내가 가진 장비가 무엇인지, 내 가방(인벤토리)에 포션은 몇 개나 있는지 확인하는 일이다.

현대 사회에서 이것을 **재무상태표(Balance Sheet)**와 **현금흐름표(Cash Flow Statement)**라고 부른다. 이름만 들어도 머리가 아프고 '로그아웃'하고 싶어지는가? 걱정 마라. 우리는 회계사가 되려는 게 아니다. 그저 내 게임 캐릭터의 현재 능력치와 골드 수급 현황을 대충이라도 파악하려는 것뿐이다.

## 📊 인벤토리 체크: "이게 다 내 장비가 아니었어?" (재무상태표)
재무상태표는 어느 한 시점(예: 오늘)의 내 재산 상태를 찍은 '스냅샷'이다. 공식은 아주 간단하다.

> **자산(내 장비) - 부채(빌린 장비) = 순자산(진짜 내 레벨)**

많은 사람이 '자본주의'라는 게임에서 흔히 범하는 실수가 있다. 번쩍이는 풀플레이트 갑옷(비싼 자동차, 아파트)을 입고 있다고 해서 그게 모두 자신의 레벨이라고 착각하는 것이다. 하지만 그 갑옷의 80%가 '대출'이라는 이름의 렌탈 장비라면, 당신의 진짜 레벨은 생각보다 낮을 수 있다.

- **자산**: 현금, 예금, 주식, 부동산, 보험 해약환급금 등 '돈이 되는 모든 것'.
- **부채**: 주택담보대출, 마이너스 통장, 카드 할부, 빌린 돈 등 '언젠가 갚아야 할 모든 것'.
- **순자산**: 자산에서 부채를 빼고 남은 '진짜 내 돈'. 이 숫자가 바로 당신의 현재 실질적인 공격력이다.

### [인벤토리 체크리스트 (재무상태표)]

| 구분 | 내 인벤토리 항목 (예시) | 금액 |
| :--- | :--- | :--- |
| **현금성 자산** | 입출금 통장, 비상금, CMA, 파킹통장 | |
| **투자 자산** | 주식, 코인, 펀드, ETF, ISA | |
| **연금/저축** | 퇴직연금(DC/IRP), 개인연금, 청약 | |
| **실물 자산** | 거주용 집(전세금), 자동차, 기타 부동산 | |
| **(A) 자산 총계** | **모든 자산의 합** | **원** |
| **단기 부채** | 카드 할부금, 마이너스 통장, 신용대출 | |
| **장기 부채** | 주택담보대출, 전세자금대출 | |
| **(B) 부채 합계** | **갚아야 할 총 금액** | **원** |
| **진짜 내 레벨** | **순자산 (A - B)** | **원** |

## 💰 골드 로그 확인: "골드가 어디로 새고 있지?" (현금흐름표)
재무상태표가 정지된 사진이라면, 현금흐름표는 한 달간 내 주머니에서 골드가 어떻게 움직였는지 보여주는 '동영상'이다.

> **수입(입금된 골드) - 지출(소모된 골드) = 잉여 현금(다음 스테이지를 위한 총알)**

아무리 고레벨 사냥터에서 골드를 많이 벌어와도(고소득), 사냥 나갈 때마다 비싼 포션을 들이붓고(과소비) 마을에 돌아와 흥청망청 써버린다면 통장은 언제나 0원에 수렴한다.

<div align="center">

**중요한 건 '얼마나 버느냐'보다 '내 손에 얼마가 남느냐'이다.**

</div>

이 남은 돈이야말로 나중에 우리가 다룰 '복리의 눈덩이(7장)'를 만들 가장 소중한 재료가 된다.

### [골드 로그 (현금흐름표)]

| 구분 | 골드 흐름 항목 (예시) | 금액 |
| :--- | :--- | :--- |
| **주수입** | 본인 급여, 사업 소득 | |
| **부수입** | 이자, 배당금, 부업, 당근 판매 수익 등 | |
| **(A) 총 수입** | **입금된 모든 골드** | **원** |
| **고정 지출** | 월세/대출이자, 보험료, 통신비, 정기구독 | |
| **변동 지출** | 식비, 교통비, 취미/여가, 경조사 | |
| **(B) 총 지출** | **소비된 모든 골드** | **원** |
| **잉여 현금** | **저축 및 투자 가능 금액 (A - B)** | **원** |

## 🧐 왜 '대충'이라도 적어야 할까?
처음부터 소수점 단위까지 맞출 필요는 없다. 숫자를 마주하는 게 두려워 아예 눈을 감아버리는 것이 가장 위험하다. 

1. **객관적 거리두기**: 숫자로 내 상태를 보는 순간, 막연한 불안감은 '해결해야 할 문제'로 바뀐다.
2. **리프레이밍**: "나는 왜 이렇게 돈이 없지?"라는 자책 대신, "아, 여기서 골드가 새고 있었구나! 여기만 막으면 다음 달엔 총알이 더 생기겠네?"라는 전략적 사고를 하게 된다.
3. **세이브 포인트**: 현재 상태를 기록하는 것은 게임의 '세이브'와 같다. 나중에 자산이 불어났을 때, 이때 적어둔 숫자를 보며 자신의 성장을 체감할 수 있는 소중한 데이터가 된다.

[퀘스트 완료 레벨업 질문]
• 당신의 인벤토리에서 가장 큰 비중을 차지하는 '자산'과 가장 의외로 많았던 '부채'는 각각 무엇인가?
• 지난 한 달간의 지출 중, 당신의 북극성(목표)으로 가는 데 전혀 도움이 되지 않았던 '버려진 골드'는 무엇인가?
• 지금 당장 모든 수입이 끊긴다면, 당신의 현재 인벤토리(비상금)만으로 몇 달을 버틸 수 있는가? (이것이 당신의 '생존 체력'이다.)

---
> [!TIP]
> **완벽한 장부는 필요 없다.**
> 가계부를 매일 쓸 자신이 없다면, 일주일에 한 번이라도 통장 잔고를 확인하고 큰 지출 항목만 적어보자. 복잡한 엑셀 대신 스마트폰 메모장이라도 좋다. '인식'하는 순간, 당신의 돈은 당신의 통제를 받기 시작한다.
