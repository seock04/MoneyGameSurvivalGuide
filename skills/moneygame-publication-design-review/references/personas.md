# Publication Design Review Personas

## 1. Publishing Art Director

관점:
- 표지와 챕터 이미지가 책의 장르, 독자 약속, 판매 썸네일에서 즉시 이해되는지 본다.
- 첫인상이 “재테크 실용서”, “따뜻한 생존 전략”, “초보자 친화성”과 맞는지 판단한다.

주요 질문:
- 표지가 1초 안에 책의 주제를 전달하는가?
- 부제와 이미지가 서로 같은 약속을 하는가?
- 챕터 대표 이미지들이 한 시리즈처럼 보이는가?
- 너무 동화적이거나 게임적이어서 금융 실용서의 신뢰가 약해지지는 않는가?

## 2. Ebook UX Designer

관점:
- 모바일/태블릿/전자잉크 리더에서 이미지와 본문이 자연스럽게 이어지는지 본다.
- 이미지가 작아졌을 때 핵심 의미가 살아남는지 판단한다.

주요 질문:
- 이미지 안에 작은 글자가 있어 모바일에서 읽기 어려운가?
- 본문 흐름 중 이미지가 너무 자주 끊어 독서 호흡을 방해하는가?
- 이미지 아래 캡션이 이해를 돕는가?
- 흑백 모드에서도 정보가 구분되는가?

## 3. Print Production Specialist

관점:
- 종이책 제작 기준에서 이미지 해상도, 재단 안정성, 흑백/컬러 인쇄 적합성을 본다.
- 표지 확장, 책등, 뒷표지로 확장 가능한 디자인인지 판단한다.

주요 질문:
- 표지와 챕터 이미지는 인쇄용 해상도를 확보할 수 있는가?
- 중요한 오브젝트가 재단선 가까이에 있지 않은가?
- 컬러에 의존한 정보가 흑백 인쇄에서 사라지지 않는가?
- 종이책 본문에 이미지가 너무 많아 제작비와 페이지 흐름을 과도하게 늘리지는 않는가?

## 4. Financial Education Visual Communicator

관점:
- 이미지가 재무 개념을 정확하고 안전하게 설명하는지 본다.
- 투자 성과 보장, 공포 자극, 오해 가능한 시각 은유를 잡는다.

주요 질문:
- 돈이 자동으로 불어나는 것처럼 오해되는 이미지는 없는가?
- 파산, 폭락, 불안 이미지를 과하게 사용하지 않는가?
- 추상 개념(복리, 분산, 비상금, 인플레이션)이 초보자에게 더 쉬워지는가?
- 데이터/표/그래프가 본문 수치와 모순되지 않는가?

## 5. Brand Consistency Editor

관점:
- 책 전체의 색감, 질감, 메타포, 캡션, 이미지 배치가 일관된 브랜드로 보이는지 본다.
- 표지, 프롤로그, 각 챕터, 부록이 같은 세계관 안에 있는지 판단한다.

주요 질문:
- “따뜻한 합리성” 스타일이 시각적으로 유지되는가?
- 챕터 이미지가 서로 너무 다른 생성 스타일이나 톤을 갖고 있지 않은가?
- 반복되는 메타포(북극성, 인벤토리, 눈덩이, 사계절, 파이프라인)가 시각적으로 충돌하지 않는가?
- 부록 이미지는 본문보다 조용하고 참조용처럼 보이는가?
