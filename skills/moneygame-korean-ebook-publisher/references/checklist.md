# Korean Ebook Publishing Checklist

## 1. Manuscript Package

- 표지 이미지 최종본
- 전자책용 제목/부제
- 저자명/필명
- 출판사명 또는 발행처
- 판권면
- 목차
- 본문
- Sources
- 투자/재무 면책 문구
- 저자 소개
- 책소개 400자/1000자 버전
- 카테고리/키워드

## 2. EPUB Build

- 통합 원고 생성
- 이미지 경로 정리
- alt text 점검
- 표를 모바일에서 읽을 수 있게 단순화
- CSS 작성
- metadata YAML 작성
- cover image 지정
- EPUB3 생성
- EPUBCheck 검증
- 뷰어별 QA

## 3. Domestic Publication Admin

- 출판 경로 결정: 직접 출판사 vs 대행 플랫폼
- 발행자번호 필요 여부
- ISBN 신청
- EPUB/PDF 별도 ISBN 여부
- 가격/발행일/판형 정보 확정
- 전자책 납본 준비

## 4. Platform Listing

- 상품명
- 부제
- 저자소개
- 책소개
- 목차
- 미리보기 범위
- 카테고리
- 키워드
- 가격
- DRM/TTS/인쇄 제한
- 성인/청소년 유해성 해당 없음 확인

## 5. Launch QA

- EPUBCheck error 0
- 내부 링크 정상
- 외부 링크 정상 또는 링크 정책 명시
- 이미지 누락 0
- 표/그래프 모바일 가독성 확인
- 본문 검색/TTS 흐름 확인
- 투자 조언 오해 문구 제거
- 납본/정산/세금 관련 후속 작업 기록
