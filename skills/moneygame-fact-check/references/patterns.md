# Patterns (Claim Extraction)

아래 패턴은 "검증이 필요한 정량 주장"을 빠르게 찾기 위한 보조 도구다.

## 숫자/기간/배수 후보 찾기

```sh
rg -n "(19|20)\\d{2}년|\\b(19|20)\\d{2}\\b|~\\s*(19|20)\\d{2}|\\d+%|\\d+배|\\d+원|\\d+만원|\\d+억" chapters/*.md
```

## 강한 사실 주장 표현 찾기

```sh
rg -n "통계적으로|증명된 사실|필연적으로|반드시|항상|유일한|결과를 초래" chapters/*.md
```

## 표/수치가 있는 구간 찾기

```sh
rg -n "^\\|.*\\|$" chapters/*.md
```
