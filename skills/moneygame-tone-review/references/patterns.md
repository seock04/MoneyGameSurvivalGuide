# Patterns

이 파일은 `moneygame-tone-review`에서 사용하는 자동 탐지 패턴 모음이다.

## 1) 서술 존댓말(격식체) 탐지

```sh
rg -n "습니다|합니다|됩니다|입니다" chapters/*.md
```

## 2) 요청/명령 어미 탐지

```sh
rg -n "세요|하십시오|바랍니다|드립니다" chapters/*.md
```

## 3) 질문 어미(정책 결정 필요)

질문 어미는 독자 인터랙션을 위해 의도적으로 쓴 것일 수 있다.
프로젝트에서 "질문은 존댓말 허용"인지 "질문도 평어체로 통일"인지 결정한 뒤 해석한다.

```sh
rg -n "인가요|나요\\?|나요\\.|나요$|신가요|하시나요|느끼시나요" chapters/*.md
```

## 4) 게임 메타포 신호(참고)

```sh
rg -n "부의 게임|인벤토리|로그인|아이템|퀘스트|자동 사냥|고인물" chapters/*.md
```
