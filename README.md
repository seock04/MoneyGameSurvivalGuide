# MoneyGameSurvivalGuide

『부의 게임, 대충 뛰어들어 끝까지 살아남기』 집필용 저장소

**부의 게임, 대충 뛰어들어 끝까지 살아남기**는 재테크를 완벽하게 준비한 뒤 시작하려는 사람보다, 지금의 삶을 무너뜨리지 않는 선에서 작게 시작하고 오래 지속하려는 사람을 위한 실전 가이드입니다.

우리는 완벽한 때를 기다리거나 화려한 지표를 분석하느라 종종 '심리적 지연'에 걸리곤 합니다. 이 책은 대박보다 생존을 먼저 두고, 예금·투자·자동화·분산·심리 관리라는 기본기를 내 삶에 맞게 연결하는 방법을 다룹니다.

핵심은 단순합니다. 작은 돈 관리 시스템을 만들고, 무리한 욕심으로 파산하지 않게 지키며, 시간이 힘을 보탤 수 있을 만큼 오래 남아 있는 것입니다.

---
