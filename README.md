# MoneyGameSurvivalGuide

『부의 게임, 대충 뛰어들어 끝까지 살아남기』 집필용 저장소

**부의 게임, 대충 뛰어들어 끝까지 살아남기**는 재테크라는 인생의 긴 무한 게임에서 포기하거나 무너지지 않고 끝까지 살아남아 경제적 자유를 얻기 위한 실전 가이드입니다.

우리는 완벽한 때를 기다리거나 화려한 지표를 분석하느라 종종 '심리적 지연'에 걸리곤 합니다. 이 책은 기계처럼 차가운 이성을 추구하다 좌절하는 대신, 내 가족의 일상을 지키고 편안하게 꿀잠을 자며 길게 이기는 **따뜻하고 대충 합리적인 투자 철학**을 제시합니다.

거인들의 지혜(인덱스 ETF, 자산배분)에 올라타고 시간의 복리를 재무 자동화 시스템으로 굴리면서, 심리적 함정을 지혜롭게 피해가는 방법을 함께 익혀 오래 살아남는 재무 베테랑으로 성장해보세요!
