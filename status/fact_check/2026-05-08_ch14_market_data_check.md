## S&P 500 Price Index Recalculation

기존 원고는 `1,278.73 -> 6,929.12`를 사용했으나, 기준일 설명이 모호했다. 출간 전 독자가 검증하기 쉽게 `2005년 말~2025년 말`로 기준을 명확히 바꿨다.

적용값:

| 항목 | 값 |
|---|---:|
| 시작 지수 | 1,248.29 |
| 종료 지수 | 6,845.50 |
| 기간 | 약 20년 |
| 상승 배수 | 6,845.50 / 1,248.29 = 약 5.48배 |
| 연평균 상승률 | 약 8.88% |

계산:

```text
상승 배수 = 6,845.50 / 1,248.29 = 5.4839
연평균 상승률 = (5.4839 ^ (1 / 20)) - 1 = 8.8816%
```

## KRW-Adjusted S&P 500 Sense Check

원화 환산 감각은 정확한 투자 성과표가 아니라, 달러 자산과 환율 변화가 함께 작동할 때의 대략적 감각을 보여주는 보조 지표다.

적용값:

| 항목 | 값 |
|---|---:|
| S&P 500 가격지수 상승 배수 | 약 5.4839배 |
| 2005년 연평균 원/달러 환율 | 1,023.7492원 |
| 2025년 연평균 원/달러 환율 | 1,421.3963원 |
| 환율 배수 | 약 1.3884배 |
| 원화 환산 감각 상승 배수 | 약 7.61배 |
| 원화 환산 감각 연평균 상승률 | 약 10.68% |

계산:

```text
환율 배수 = 1,421.3963 / 1,023.7492 = 1.3884
원화 환산 감각 배수 = 5.4839 * 1.3884 = 7.6140
연평균 상승률 = (7.6140 ^ (1 / 20)) - 1 = 10.6829%
```

주의:
- 실제 한국 투자자의 수익률은 매수 시점, 환전 시점, 배당, 세금, 수수료, 상품 구조에 따라 달라진다.
- 원고에도 이 표가 순위표나 수익 보장표가 아니라는 설명을 유지했다.

## R-ONE Caveat

현재 원고 값:

| 비교 대상 | 시작 지수 | 종료 지수 | 상승 배수 | 연평균 상승률 |
|---|---:|---:|---:|---:|
| 서울 동남권 아파트 매매지수 | 61.6 | 215.5 | 약 3.50배 | 약 6.46% |
| 경기 아파트 매매지수 | 66.3 | 144.9 | 약 2.19배 | 약 3.99% |

판정:
- 기존 R-ONE 확인 로그의 값을 유지한다.
- 한국부동산원 R-ONE 통계 화면은 동적 페이지라 자동 검산 재현성이 낮다.
- 출간 직전에는 동일 조건으로 화면을 열어 시작월/종료월/지역/지표 기준을 다시 캡처 확인하는 것이 안전하다.
